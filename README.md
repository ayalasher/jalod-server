# jalod-api

## Getting started
- Clone the application via HTTPS or SSH
- Have ~Docker Desktop~ installed and actively running on your computer.
- Open your preferred terminal navigate to the cloned root project directory
- From inside the project directory build the docker image `docker build -t jalod-api`
- Run the docker container `docker run -p 5000:5000 jalod-api`
- Test the app on your browser 'http://localhost:5000'
